# 🏊‍♂️ 游泳游戏

基于手机方向传感器的游泳小游戏，通过“倾斜手机”与虚拟摇杆控制方向，持续前进、收集金币并躲避障碍物。

## 🎮 功能特性

- **方向控制**：支持方向传感器（DeviceOrientation），可叠加虚拟摇杆
- **游戏机制**：60 秒挑战，金币计分、生命系统、短暂无敌、连吃奖励
- **精美界面**：响应式设计，流畅动画效果

## 🚀 快速开始

### 运行项目

1. 授权方向传感器
2. 倾斜手机（左右/前后）控制方向；角色持续前进
3. 收集金币 +1 分；连续 3 枚金币 +1 命（上限 5）
4. 撞击障碍扣 1 命并短暂无敌；初始 3 命；总时长 60 秒

## 📱 使用说明

- **倾斜控制**：基于手机方向（beta/gamma）实时移动角色
- **生命与得分**：金币 +1；3 连吃 +1 命（上限 5）；撞击扣命并短暂无敌

## 🛠️ 技术栈

- **框架**：uni-app
- **语言**：Vue.js + JavaScript
- **传感器**：DeviceMotionEvent, DeviceOrientationEvent
- **样式**：SCSS + CSS3

## 🧪 调试功能

为加速开发与测试，内置一个“调试轮盘”以及若干快捷操作：

### 调试轮盘（Playing 状态左下角🛠️按钮）

| 按钮 | 功能 |
|------|------|
| 障碍+ | 立即生成一个障碍物 |
| 金币+ | 立即生成一个金币 |
| 速度+ | 增加玩家当前速度（加速） |
| 速度- | 减少玩家当前速度（减速） |
| 粒子 | 触发一次粒子效果（调试用） |
| 暂停/继续 | 切换游戏暂停与继续 |
| 结束 | 立即结束当前局 |
| 重开 | 重置并重新开始游戏 |

### 摇杆与拖动

调试轮盘内置虚拟摇杆：按住圆形摇杆区域拖动即可持续改变角色移动方向（支持 360°），松开后复位。整个轮盘可从外缘拖动移动位置以避免遮挡。

（如需键盘快捷键，可在组件中自行扩展事件监听，当前版本未内置快捷键。）
